Download Lexio

Currently, Lexio is exclusively available for macOS (Apple Silicon recommended). The app is notarized by Apple so you can install and run it safely.

Download Lexio for Mac
Requires macOS 11 or later. Supports Apple Silicon (M1/M2/M3).

Installation Steps

  1. Download the file: Click the download button above to get the Lexio-Glance-1.3.0-arm64.dmg file.
  2. Open the DMG: Double-click the downloaded file to open the installer window.
  3. Install: Drag the Lexio icon into the Applications folder shortcut shown in the window.
  4. Launch the app: Open your Applications folder (or use Spotlight) and open Lexio. Since the app is notarized by Apple, you will simply be asked if you are sure you want to open it on first launch.

Getting Started

Once you open Lexio, just log in or create a new account. The app runs in your menu bar and allows you to instantly point at and translate words across your entire screen!

Not on a Mac? You can also get the Chrome Extension or try the web app directly.
